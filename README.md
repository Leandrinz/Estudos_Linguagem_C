# 📘 Estudos de Linguagem C

Este repositório contém **anotações e práticas sobre a linguagem C**, organizadas de forma progressiva conforme o avanço dos estudos.

A ideia do projeto é reunir **resumos em Markdown + exemplos práticos em código**, facilitando tanto o aprendizado quanto a revisão de conceitos da linguagem.

Ele funciona como um **material de apoio rápido**, onde é possível revisar rapidamente conceitos básicos da linguagem C.

Por exemplo:

- Esqueceu como funciona um **laço de repetição**?
- Quer lembrar a **sintaxe de um `for`, `while` ou `do-while`**?
- Precisa rever **condicionais ou estruturas básicas da linguagem**?

Basta navegar pelas aulas e consultar os arquivos de anotação.

---

# 📚 Organização do Repositório

O repositório é dividido em **aulas**, onde cada uma aborda um conjunto de conceitos da linguagem.

Cada aula normalmente possui:

- 📄 **Arquivo de anotações em Markdown**
- 💻 **Exemplos ou práticas em código C**

Estrutura geral:

```
aula-01/
    anotacoes.md
    exemplo.c

aula-02/
    anotacoes.md
    pratica.c

aula-03/
    anotacoes.md
```

As anotações explicam:

- conceitos
- sintaxe
- exemplos de uso

Já os códigos permitem **testar e experimentar os conceitos na prática**.
