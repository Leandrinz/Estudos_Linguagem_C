# Variáveis

## 1) O que são:
Variáveis são espaços na memória onde guardamos todo tipo de dado.

## 2) Tipos:
* **char**: caracteres
* **int**: Inteiros
* **float**: Números Reais
* **double**: Número real com maior precisão

## 3) Nome da variável:
### Não pode:
* Palavras acentuadas
* Espaços (Usa-se o `_` no lugar)
* Símbolos reservados
* Começar com números
* Palavras reservadas (ex: `int`, `char`, `return`...)

### Exemplo bom:
* `Primeiro_nome`

## 4) Como declarar:
### Sintaxe:
```c
int main(){
    char letra = 'a';
    int numero = 10;
    return 0;
}
