# Operadores de Atribuição

## 1. Atribuição Simples
O operador de atribuição é representado pelo símbolo `=`.

* **Conceito:** O valor da expressão à **direita** é avaliado e armazenado no objeto (variável) à **esquerda**.
* **Exemplos:**
  ```cpp
  int x = 10;      // A variável 'x' recebe o valor literal 10
  int y = x + 5;   // A variável 'y' recebe o resultado da soma (15)
