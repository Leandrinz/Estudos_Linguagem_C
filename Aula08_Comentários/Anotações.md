# Comentário

1) **Sintaxe:**
    * `// Comentário exemplo`
    * Você coloca essas duas barras `//` antes do texto que você quer comentar.
    * **Obs:** O comentário é totalmente descartado pelo compilador durante a compilação (não gera código executável).

2) **Importância:**
    * **Documentação:** Serve para documentar o seu código, ajudando quem está vendo de fora a entender a lógica.
    * **Manutenção:** Facilita a manutenção do sistema ao longo do tempo, tanto para você quanto para outros desenvolvedores.

3) **Comentar várias linhas de uma vez (Bloco):**
    * **Sintaxe:**
        ```c
        /*
            Tudo o que estiver aqui dentro 
            será considerado um comentário,
            não importa quantas linhas existam.
        */
        ```
