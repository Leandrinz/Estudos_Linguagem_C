# Atribuições Simplificadas

Estes operadores são formas compactas de atualizar o valor de uma variável utilizando o seu próprio valor atual.

1) **Soma (`+=`):**
    * `x += z`
    * É o mesmo que: `x = x + z`
    * O valor de `z` é somado ao valor atual de `x` e o resultado é guardado em `x`.

2) **Subtração (`-=`):**
    * `x -= z`
    * É o mesmo que: `x = x - z`
    * O valor de `z` é subtraído do valor atual de `x`.

3) **Multiplicação (`*=`):**
    * `x *= z`
    * É o mesmo que: `x = x * z`
    * O valor atual de `x` é multiplicado por `z`.

4) **Divisão (`/=`):**
    * `x /= z`
    * É o mesmo que: `x = x / z`
    * O valor atual de `x` é dividido por `z`.

5) **Resto (`%=`):**
    * `x %= z`
    * É o mesmo que: `x = x % z`
    * `x` passará a valer o resto da divisão de `x` por `z`.
