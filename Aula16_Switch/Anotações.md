# Switch

1) O que é:
    .  Estrutura de controle condicional usada na programação para selecionar e executar um bloco de código específico entre múltiplas opções (casos), baseando-se no valor de uma única expressão ou variável. 

2) Sintaxe:
    switch(variavel){
        case valor1: conjunto de comandos;break;
        case valor2: conjunto de comandos;break;
        case valorn: conjunto de comandos;break;
        default: conjunto de comandos;
    }

    . Se cair no default, é que nenhum case foi satisfeito ou verdadeiro
