# Printf

## 1) Pra que serve:
Escreve informações no terminal.

## 2) Escrevendo valores de variáveis:
Para exibir o valor de uma variável, usamos os especificadores de formato:
* **%d** -> Números inteiros (`int`)
* **%c** -> Caracteres (`char`)
* **%f** -> Números reais (`float` ou `double`)

### Sintaxe:
```c
printf("letra: %c. Numero: %d", letra, numero);
